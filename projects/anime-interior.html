Project 03

Anime-Style Interior Environment

A fully-optimized cel-shaded 3D room built in Blender and rendered in Unity

// Overview

This project is a complete technical art pipeline: from a blank Blender scene to a real-time, interactive cel-shaded room running stably in Unity at 60+ FPS. The environment is inspired by "Beast of Reincarnation" and contemporary anime interior aesthetics — warm lighting, clean outlines, expressive depth through limited shading bands, and that signature "2D art brought into 3D space" quality.

The room itself is a futuristic lab / living space hybrid: modular wall panels, a central workstation with a holographic monitor (emissive material), floor-to-ceiling windows with stylized light rays, and hand-placed props each with unique toon-shaded materials. Everything in the scene was modeled, UV-unwrapped, and textured in Blender, then brought into Unity where custom HLSL toon shaders were applied.

// What I Built
  • › Full room geometry in Blender: 12 unique mesh objects, all with clean topology and efficient polygon counts (room total: under 18,000 triangles).
  • › UV unwrapping with texture atlasing — all props share two 1024×1024 texture atlases, reducing draw calls significantly.
  • › Custom Unity HLSL toon shader with configurable shadow band count (1–4 bands), rim lighting, outline width, and specular threshold.
  • › Emissive materials for the holographic display with animated UV scroll simulating hologram flicker.
  • › Baked ambient occlusion maps — adding subtle depth without real-time AO cost. Final scene: 60+ FPS on mid-tier GPU.
// What I Learned

The biggest lesson was in the art pipeline itself: Blender's coordinate system, UV space, and normal direction conventions differ from Unity's. The first import had every mesh inside-out with flipped normals. Building a reliable export checklist turned a frustrating trial-and-error process into a repeatable workflow.

Custom HLSL shaders demystified the rendering pipeline. Understanding how fragment shaders sample textures, how the dot product of the normal and light direction drives diffuse bands, and how depth-based outline detection works — this knowledge now applies to every graphics project.

// The Challenge

Getting the toon outline to look clean on complex silhouettes without expensive geometry duplication — some edge cases (curved surfaces, thin props) made the outline shader produce ugly artifacts.

// The Approach

Switched from an inverted-hull geometry outline to a screen-space post-processing outline using a Sobel edge detection filter on the depth buffer. Cleaner results at any camera distance, zero polygon overhead, and controllable via a global outline thickness parameter.
